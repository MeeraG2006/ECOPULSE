PROJECT DESCRIPTION 
[ECO PULSE IS A SMART WEB APPLICATION THAT HELPS USERS CALCULATE AND MONITOR THEIR DAILY CARBON FOOTPRINT BASED ON LIFESTYLE ACTIVITIES SUCH AS TRAVEL ELECTRICITY USAGE, FOOD COMSUMPTION . IT PROVIDES PERSONALISED SUGGESTIONS AND ALSO GIVES THE GREEN SCORE
PROBLEM STATEMENT
[People are often unaware of how their daily activities contribute to carbon emissions, creating a need for a simple platform that helps individual track , understand and reduce the personal carbon footprint.
SOLUTION
[Ecopulse provides a smart web platform that calculates user's carbon footprint based on daily activities and visualize their environmental impact , and offers personalised suggestiona and interacticve chat bot to encourage sustainability and eco friendly lifestyle choices.

FEATURES
Feature 1: User Authentication System – Allows users to securely sign up, log in, and maintain personalized carbon footprint records.
Feature 2: Carbon Footprint Calculator – Calculates carbon emissions based on transportation, electricity usage, food habits, and lifestyle activities.
Feature 3: Weekly Impact Analysis & Visualization – Displays emission data using charts and reports to help users track environmental impact over time.
Feature 4: Smart Eco Chatbot – Provides sustainability tips, answers user queries, and suggests eco-friendly actions based on user activity.
Feature 5: green score
